# Description du programme
### `#include <stdio.h>`
- Fichier externe inclu
- Fichier header
- Contient des déclarations de focntions comme `printf`
### `int maint() {`
- `main` : Fonction principale d'un programme C
    - Point d'entrée
- `int` : La fonction retournera un entier en cas de succès du programme
- Ne prends pas d'arguments : `()`
### `printf("Hello World!\n);`
- `printf` : Fonction qui affiche du texte à l'écran
- `"..."` : Chaîne de caractères
- `\n` : Retour à la ligne
- `;` : Fin de l'instruction
### `return 0;`
- La fonction `main` doit retourner un entier
- Elle retourne `0` si le programme s'est terminé correctement
### `}`
- Fin de la fonction `main`
